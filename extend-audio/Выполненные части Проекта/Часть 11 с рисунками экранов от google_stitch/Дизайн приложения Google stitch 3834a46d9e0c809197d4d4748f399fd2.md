Экран профиля (в начальной версии просто локальная статистика)

Приложение будет выполнено в тёмной теме. Основной фон — тёмно-синий/чёрный, акцент — оранжевый. Карточки с закруглёнными углами, минималистичные иконки, крупные кнопки управления. Шрифт — без засечек (Inter или аналог). Интерфейс современный минималистичный, без лишних элементов.

| Экран по ТЗ | Что показывает дизайн |
| --- | --- |
| Главный экран
(хотбар) | Навигация через хотбар: 4 иконки внизу |
| Библиотека треков | Список загруженных файлов, кнопка загрузки |
| Библиотека пресетов | Список сохранённых пресетов, кнопка создания |
| Экран плеера | Кнопки управления, прогресс, информация о треке |
| Экран эквалайзера | Ползунки частот, кнопка «Сохранить пресет» |
| Экран профиля | Статистика: количество треков, пресетов |

Экраны сгенерированы с помощью Google Stitch. На основе описания ТЗ и пользовательских сценариев были созданы макеты основных экранов приложения.
