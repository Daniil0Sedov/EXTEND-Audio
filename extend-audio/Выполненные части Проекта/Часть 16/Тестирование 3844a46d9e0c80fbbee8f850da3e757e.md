# Тестирование EXTEND Audio

## 1. Цель тестирования

Цель тестирования — проверить, что MVP мобильного приложения EXTEND Audio соответствует требованиям, пользовательским сценариям и техническому заданию. В рамках проверки оцениваются основные экраны, навигация, импорт локальной музыки, воспроизведение треков, работа пресетов, эквалайзера и сохранение данных.

## 2. Ссылка на сборку / репозиторий

- Репозиторий проекта: https://github.com/Daniil0Sedov/EXTEND-Audio/tree/main/extend-audio
- APK-сборка: `android-app/app/build/outputs/apk/debug/app-debug.apk`

## 3. Тест-план

| Поле | Ответ |
| --- | --- |
| Название проекта | EXTEND Audio |
| Цель тестирования | Проверить готовность MVP мобильного приложения к демонстрации и защите |
| Что проверяется | Импорт папки с музыкой, библиотека треков, поиск, экран плеера, пресеты, эквалайзер, навигация, сохранение данных |
| Что не входит в тестирование | Серверная часть, авторизация, облачные функции, социальные функции, онлайн-синхронизация |
| Основные риски | Нестабильная работа эквалайзера на разных устройствах, проблемы с локальными обложками, различия в поведении Android API |
| Критерии завершения | Все Must Have сценарии MVP проверены, критические ошибки отсутствуют, результаты подтверждены скриншотами или логами |

## Итоговый отчет о готовности MVP

В рамках тестирования приложения EXTEND Audio были проверены основные пользовательские сценарии MVP: импорт локальной папки с музыкой, отображение библиотеки треков, поиск, переход на экран плеера, воспроизведение аудио, работа с пресетами, навигация между основными экранами и сохранение данных. Тестирование показало, что базовый сценарий использования приложения реализован и может быть продемонстрирован на защите.

По итогам проверки были зафиксированы результаты по модульным, интеграционным и ручным UI-проверкам. Найденные ошибки оформлены в виде баг-репортов и, при наличии исправлений, повторно проверены. Таким образом, раздел тестирования подтверждает фактическую работоспособность MVP и показывает степень его готовности к дальнейшей доработке.
